## @Paracraft

## 参数说明

```
- styleID :  //样式ID

# preview
- src : // Paracraft世界预览图
- link : // 预览图链接

# worldName
- name : // 世界名称
- link : // 世界链接

# author
- text : // 作者名称
- desc : // 作者label 描述
- link : // 作者链接

# version
- text : // 版本号
- desc : // 版本label 描述
- link : // 版本链接

# updateTime
- text : // 更新时间
- link : // 更新链接

# viewTimesImg
- src : // 图片地址
- link : // 图片链接

# viewTimes
- text : // 浏览次数
- desc : // 浏览次数label
- link : // 浏览次数链接

# size
- text : // 世界大小
- desc : // 世界大小label 描述
- link : // 世界大小链接

# download
- name : // 下载按钮名称
- link : // 下载按钮链接

# enter
- name : // 进入按钮名称
- link : // 进入按钮链接

# paracraft_info
- text : // 未安装Paracraft客户端提示文字
- link : // 下载Paracraft客户端链接
```
